# YouTube 다운로더

간단한 GUI 기반 YouTube 비디오 다운로더입니다.

## 기능

- YouTube URL을 통한 비디오 다운로드
- 사용 가능한 모든 화질 표시
- 최고 화질 선택 가능
- 비디오+오디오 통합 또는 별도 다운로드
- 오디오만 다운로드
- 다운로드 진행 상황 표시
- 저장 위치 선택

## 설치 방법

1. Python 3.7 이상이 설치되어 있어야 합니다.

2. 필요한 패키지 설치:
```bash
pip install -r requirements.txt
```

## 사용 방법

1. 프로그램 실행:
```bash
python main.py
```

2. YouTube URL 입력

3. "정보 가져오기" 버튼 클릭

4. 원하는 화질 선택:
   - **비디오+오디오 (권장)**: 비디오와 오디오가 이미 결합된 포맷
   - **비디오 전용 (최고 화질)**: 최고 화질의 비디오 (오디오는 자동으로 병합됨)
   - **오디오 전용**: 음악만 다운로드

5. 저장 위치 선택 (기본값: 다운로드 폴더)

6. "다운로드" 버튼 클릭

## 참고 사항

- 비디오 전용 포맷을 선택하면 자동으로 최고 품질의 오디오가 병합됩니다.
- 다운로드 속도는 인터넷 연결 속도에 따라 달라집니다.
- 일부 비디오는 저작권으로 인해 다운로드가 제한될 수 있습니다.

## 음질 정보

- **간편 선택**을 사용하면 자동으로 최고 품질 오디오(Opus 또는 AAC)가 선택됩니다.
- Opus 코덱은 YouTube에서 제공하는 최고 음질 포맷입니다 (최대 160kbps).
- 다운로드 완료 후 폴더 자동 열기 옵션을 활성화하면 편리합니다.

## 문제 해결

### ffmpeg 오류
비디오와 오디오 병합 시 ffmpeg가 필요합니다:
- Windows: https://www.gyan.dev/ffmpeg/builds/ 에서 다운로드 후 PATH에 추가
- macOS: `brew install ffmpeg`
- Linux: `sudo apt-get install ffmpeg`

### yt-dlp 업데이트
```bash
pip install --upgrade yt-dlp
```
